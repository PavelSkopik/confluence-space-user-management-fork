Gary S. Weaver added a comment - 07/Apr/09 9:01 AM

Issa,

There used to be a bug where refreshing didn't work:

  • https://support.atlassian.com/browse/CSP-10371
  • https://jira.atlassian.com/browse/CONF-9035

Even though there is no version# listed where this was to be fixed, I believe it was fixed either in 2.6 or earlier (possibly 2.5.7?) or it may be possible that it wasn't fixed at all, but that is what I remember.

To work around this for those versions of Confluence that it affected, we added a removable hack that would redirect in order to cause the refresh to occur so that after add/removing a user/group, they would show up on the page without having to reload the page.
