Welcome to the Custom Space User Management project

Introduction

Welcome to the C.S.U.M. Confluence plugin project maven documentation. This contains technical details about the plugin project itself for those interested. You are probably more interested in the main project page. Rajendra also took a really great team picture if you're interested in seeing our ugly mugs.

Description

This plugin gives Space Administrators ability to manage users and user-groups associated to their Wiki Spaces.
